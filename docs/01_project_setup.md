## 1.2 Create the Flutter Project

```bash

flutter create math_kingdom_builder \
  --org com.yourname.mathkingdom \
  --platforms ios,android,web

cd math_kingdom_builder

flutter doctor
```

## 1.5 Hive Setup (local database)

In `main.dart`:

```dart
import 'package:hive_flutter/hive_flutter.dart';

void main() async {
  WidgetsFlutterBinding.ensureInitialized();
  await Hive.initFlutter();

  // Register adapters (generated by hive_generator)
  Hive.registerAdapter(ChildProfileAdapter());
  Hive.registerAdapter(KingdomStateAdapter());
  Hive.registerAdapter(LessonProgressAdapter());

  // Open boxes
  await Hive.openBox<ChildProfile>('profiles');
  await Hive.openBox<KingdomState>('kingdoms');
  await Hive.openBox<LessonProgress>('progress');

  runApp(
    const ProviderScope(
      child: MathKingdomApp(),
    ),
  );
}
```
